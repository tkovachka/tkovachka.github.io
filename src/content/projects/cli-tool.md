## Overview

A CLI tool that automates several repetitive tasks in my daily workflow — scaffolding new projects,
managing environment configs, and syncing dotfiles across machines.

## Features

- Zero-dependency core with optional plugin system
- Interactive prompts via `@inquirer/prompts`
- Fully typed with TypeScript and published as ESM

## Status

Completed and published to npm. Actively maintained.
