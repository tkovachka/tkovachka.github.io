This is a computer graphics project rendering a simple 3D Quidditch Field. Built entirely in C++ using OpenGL and GLSL shaders, utilizing CMake as the build system.

## Overview

I am only including this project in my portfolio to say that not all of the projects that I take on are a success, but still matter nonetheless.

This project is an example of a half-assed attempt to make a quick video-game-like environment where the user can "fly on a broom" around the Quidditch field and explore. It definitely had potential to be turned into something much larger, even a fully-fledged game where you could play Quidditch like you play football, but I simply could not get the hang of OpenGL and GLSL shaders, and had more problems with CMake than I care to admit...

The whole experience turned out to be me experimenting in Blender, trying to be wannabe graphic designer, but I quickly learnt that 3D modelling is not for me.

And if you've read my story you know that the promise of game development was what got me into programming in the first place, so this project is a bit of a sore spot for me, especially with having to come to terms with the fact that I will likely never be a game developer. But hey, at least I can say that I tried, right? And I guess it's not all bad, I did learn a thing or two about OpenGL and GLSL shaders, and CMake, even if I did end up hating them.
