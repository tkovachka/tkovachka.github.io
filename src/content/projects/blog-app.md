A DevOps pipeline showcasing continuous deployment of a blog-post application using modern container orchestration tools. The project demonstrates the usage of Docker for containerization, Kubernetes for orchestration, and ArgoCD for GitOps-style deployments.

## Overview

This application is not mine to begin with. It is a forked repository that I used as a baseline to learn about Docker and Kubernetes.

I'm not gonna lie - it was tough getting the hang of all of it, especially trying to grasp the concept of containerization and orchestration in general. That's why I chose such a simple app to begin with - so I could have the capacity to focus on the DevOps pipeline itself, rather than the application code. I definitely needed a project solely for the purpose of experimenting with DevOps tools.

I have marked the project as *Completed* for now, while I am working on other priorities, but I am keeping it in mind for future DevOps experiments. It would be nice to continue this flow with a more complex application, and I have a few ideas in mind...
